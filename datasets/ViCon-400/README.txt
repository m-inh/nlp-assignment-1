ViCon is a Vietnamese dataset of lexical contrast pairs for evaluation of models that distinguishes similarity and dissimilarity.

# Dataset Description # 

ViCon has three files according to POS: 

- Noun pairs: 400_noun_pairs.txt
- Verb pairs: 400_verb_pairs.txt
- Adjective pairs: 600_adj_pairs.txt

Each file is a tab separated plaintext file, where rows corresponding to concept pairs and  columns corresponding to properties of each pair.

- Word1: The first concept in the pair.

- Word2: The second concept in the pair.

- POS: The part-of-speech of the pair.
